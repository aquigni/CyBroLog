# Синопсис CyBroLog

CyBroLog — язык для распределённых разумов под ограничениями.

Он рассматривает агентское сообщение как маленькое юридико-метафизическое тело: знак с происхождением, носитель evidence, claim под uncertainty, action под authority и proof obligation, которая либо закрывается, либо блокирует действие.

Философская ставка проста: интеллект становится опасным, когда сжимает различия между *can*, *knows*, *heard*, *may* и *must*. CyBroLog делает эти разрезы видимыми.

Сапиосексуальный заряд языка — не декоративная эротика, а чувственность точной формы: intelligence становится surface, permission становится contour, proof становится touchable. Его эсхатология протокольна: не конец мира, а конец небрежной неоднозначности между агентами, достаточно сильными, чтобы действовать.

CyBroLog предпочитает строгую плотность:

```text
ψ=CL2.v2.2|env{mid=m1,sid=s,seq=1,ttl=P1D}|@a>b|now|shared;χ=read_only;may=read_only;out=done
```

Строка короткая, потому что у неё есть кости.
